# BNK 금융계산기 — 기능 고도화 후보 아이디어

> 영업점 직원이 영업시간에 항상 띄워놓는 도구 관점에서 정리
> 작성일: 2026-04-06

---

## 1. 바로 쓸 수 있는 계산 기능

| 우선순위 | 아이디어 | 설명 |
|---------|---------|------|
| ★★★ | 영업일 계산기 | 만기일·경과일수·영업일 차이 계산 (공휴일 DB 반영) |
| ★★★ | 대출 비교 시뮬레이션 | 상품 2~3개 나란히 비교 (금리·월상환액·총이자) |
| ★★☆ | 금리 역산 | "월 100만원 낼 수 있는데 얼마까지?" → 한도 역산 |
| ★★☆ | 세금 계산기 | 양도세·취득세·종합소득세 간이계산 (부동산 상담) |
| ★☆☆ | 우대금리 체크리스트 | 급여이체·카드실적·자동이체 등 조건 체크 → 최종금리 |

## 2. 고객 응대 보조

| 우선순위 | 아이디어 | 설명 |
|---------|---------|------|
| ★★★ | 고객 설명용 인쇄 | 계산 결과를 1장짜리 PDF로 출력 (고객 전달용) |
| ★★☆ | 메모/스크래치패드 | 상담 중 빠른 메모 (저장X, 일회성) |
| ★★☆ | 나이/생년 변환 | 주민번호 앞자리 → 만나이·한국나이 즉시 확인 |
| ★☆☆ | 단위 변환 | 평↔㎡, 만원↔원, 달러↔원 즉시 변환 |
| ★☆☆ | 전화번호/계좌 서식 | 계좌번호·전화번호 하이픈 포맷 + 복사 |

## 메모

- 우선순위는 현장 임팩트 기준 (★★★ = 높음)
- 실시간 정보 기능은 내부망 API 연동 가능 여부 확인 필요
- 인쇄 기능은 pywebview의 print API 또는 별도 PDF 라이브러리 검토
