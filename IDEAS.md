> 최종 정리: 2026-04-07

---

## 구현 예정

| 우선순위 | 아이디어 | 설명 |
|---------|---------|------|
| ★★★ | 기본계산기 | 사칙연산 + M+ 메모리 기능 (항상 띄워놓는 도구에 필수) |
| ★★★ | 메모 + TODO 리스트 | 계산 메모와 별도 독립적인 메모/할일 관리 기능 |
| ★★★ | 규정 즐겨찾기 링크 | URL 직접 등록/삭제 가능한 북마크 (행내 규정 바로가기) |
| ★★★ | 대출 비교 시뮬레이션 | 상품 2~3개 나란히 비교 (금리·월상환액·총이자) |
| ★★☆ | 나이/연령구분 | 주민번호 앞자리 → 14세 미만·미성년자·고연령자(금융취약) 구분 표시 |
| ★★☆ | 피젯스피너 | 점심 사다리/룰렛 등 간단한 미니게임 (가볍게 구현, 무거워지면 X) |

## 보류/제외

| 아이디어 | 제외 사유 |
|---------|----------|
| 영업일 계산기 | 공휴일 DB 매년 유지보수 부담, 대출은 달력일 기준이라 활용도 낮음 |
| 고객 설명용 인쇄 | 실효성 낮음 |
| 금리 역산 | 대출 비교 기능에 포함 가능 |
| 세금 계산기 | 변수 과다, 간이 계산 시 오해 소지 |
| 우대금리 체크리스트 | 상품별 상이, 유지보수 힘듦 |
| 실시간 환율/기준금리 | 내부망 API 연동 불가 |
| 영업점 공지 위젯 | 인트라넷과 중복 |
| 자주 쓰는 조건 템플릿 | 이력 기능과 겹침 |
| 단위 변환/전화번호 서식 | 사소함 |
| 타이머/알림 | OS 기본 기능으로 대체 |
| 메모/스크래치패드 | 메모+TODO로 통합 |
| M+ 메모리 | 기본계산기에 포함 |
