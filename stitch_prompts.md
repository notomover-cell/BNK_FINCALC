## 프롬프트 1: 메인 프레임 + 대출 계산기 탭

```
한국 은행 창구 직원용 금융계산기 데스크톱 앱. 폭 440px, 높이 자동. 한국어 UI.

스타일: 토스 앱 스타일 클린 미니멀. 흰 배경(#FFFFFF), 포인트 컬러 블루(#3182F6), 
보조 레드(#F04452 세금/이자 표시용), 텍스트 다크그레이(#333333), 
서브텍스트 (#8B95A1), 구분선 (#F2F4F6). 라운드 카드, 부드러운 그림자.

[상단 탭 바]
가로 스크롤 가능한 탭: 예금 | 적금 | 대출 | 주담대종합 | LTV | DTI | DSR | 중도상환 | 환전 | 이력
현재 활성 탭: "대출" (블루 밑줄 + 블루 텍스트)

[대출 계산기 - 입력 영역]
카드 형태, 라운드 8px, 패딩 20px

1. "대출금액" 라벨 + 금액 입력 필드 (우측에 "원" 표시, 천단위 콤마)
   아래에 빠른 금액 버튼 가로 나열: +10만 | +100만 | +1,000만 | +1억

2. "대출기간" 라벨 + 숫자 입력 필드
   우측에 년/월 토글 버튼 (pill shape, 선택된 쪽 블루 배경)
   아래에 빠른 기간 버튼: +1년 | +5년 | +10년

3. "대출금리 (연)" 라벨 + 소수점 입력 필드 (우측에 "%" 표시)

4. "상환방법" 라벨 + 3개 토글 버튼 그룹 (pill shape):
   원리금균등 | 원금균등 | 만기일시
   선택된 항목 블루 배경

[하단 버튼]
"초기화" 버튼 (회색 아웃라인) | "계산하기" 버튼 (블루 배경, 흰 텍스트, 풀 너비)

[계산 결과 영역]
별도 카드, 상단에 "계산 결과" 헤더 + 접기/펼치기 아이콘

- 대출원금: 300,000,000원 (우측 정렬)
- 총 대출이자: 158,536,200원 (레드 색상)
- 총 상환금액: 458,536,200원
- 구분선
- ★ 매월 상환금액: 1,273,712원 (큰 폰트, 블루 색상, 강조)
- [월별 상환 스케줄 보기] 텍스트 버튼 (블루)

우측 하단에 [M+] 버튼 (메모리 저장)
```

## 프롬프트 4: DSR 계산기 탭

```
동일한 금융계산기 앱. 활성 탭: "DSR". 동일 디자인 시스템.

[DSR 계산기 - 입력 영역]

섹션1 "소득 정보" (서브헤더, 회색 배경 라벨)
1. "연소득" + 금액 입력, 빠른 금액 버튼

섹션2 "신규 주택담보대출" (서브헤더)
2. "대출금액" + 금액 입력, 빠른 금액 버튼
3. "대출기간" + 숫자 입력, 년/월 토글
4. "대출금리(연)" + 소수점 입력 (%)
5. "상환방법" 토글: 원리금균등 | 원금균등 | 만기일시

섹션3 "보유 대출" (서브헤더)
6. "보유대출 연이자 상환액" + 금액 입력
7. "보유대출 연원금 상환액" + 금액 입력

섹션4 "스트레스 DSR" (서브헤더, 우측에 info 툴팁 아이콘)
8. "지역 구분" 토글: 수도권·규제지역 | 지방 | 기타(전국)
9. "금리 유형" 토글: 변동형 | 혼합형 | 주기형 | 순수고정형

[계산 결과]
- 신규대출 연원리금: 15,284,520원

- 구분선
- 스트레스 DSR 영역 (연한 오렌지 배경 카드):
  - "수도권·규제지역, 변동형"
  - 가산금리: 3.0% x 100% = +3.0%p
  - 심사금리: 3.5% + 3.0% = 6.5%
  - 스트레스 DSR: 45.1% (오렌지 강조, 큰 폰트)
- 회색 안내 문구: "규정 기준일: 2026-04-05 | 예상 수치이며 실제와 다를 수 있습니다"

## 프롬프트 5: LTV 계산기 탭

```
동일한 금융계산기 앱. 활성 탭: "LTV". 동일 디자인 시스템.

[LTV 계산기 - 입력 영역]

1. "대출목적" 토글: 신규주택구입 | 생활안정자금
2. "보유 주택 수" 토글 (2줄 가능): 생애최초 | 서민실수요 | 무주택 | 1주택(처분조건부) | 2주택이상
3. "규제지역 여부" 토글: 규제지역(수도권) | 비규제지역
4. "담보주택 시세" + 금액 입력, 빠른 금액 버튼
5. "보유 대출 잔액" + 금액 입력 (조건부 표시)
6. "임차보증금" + 금액 입력 (조건부 표시)
7. "소액임차보증금" + 금액 입력 (기본값 채워짐, 수정 가능, 우측에 "정책변수" 뱃지)

- 절대한도: 600,000,000원 (해당 시 표시)
- 구분선
- ★ 최대 대출 가능 금액: 200,000,000원 (블루 강조, 큰 폰트)
- 조건 요약 뱃지: "신규주택구입 · 무주택 · 규제지역"
- 안내 문구 (2줄):
  "규정 기준일: 2026-04-05"
  "입력 정보 기반 예상 금액입니다. 정확한 금액은 금융사 확인 필요."

- 경과일수: 365일 / 잔존일수: 730일
- 구분선
- ★ 예상 중도상환수수료: 193,333원 (블루 강조)
  (또는 "부과기간 경과 - 수수료 없음" 그린 배지)
- 산식 표시: "50,000,000 x 0.58% x (730/1,095)"
