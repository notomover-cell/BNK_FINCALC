## 1. 예적금 이자 계산

### 공식: ✅ 정확

부동산계산기.com의 예적금 이자 계산기도 동일한 단리/복리 공식을 사용한다. 적금 단리 공식 `월납입액 × (연이율/12) × n(n+1)/2`는 동 사이트에서도 동일하게 적용된다.

### 수정 필요 사항

**과세 분류명을 "기타과세"로 변경 고려.** 부동산계산기.com은 "비과세 / 일반과세 / 기타과세"로 분류한다. 사용자 설계의 "세금우대 1.4%"는 상호금융 조합원 한정 저율과세(농특세만)이며, 실제로는 "세금우대종합저축 9.5%"와 "상호금융 저율과세 1.4%"가 별도로 존재한다. 우리은행 금융계산기도 세금우대 9.5%를 적용하고 있어, **1.4%만 표기하면 불완전**하다.

**권고:** 과세 옵션을 다음과 같이 확장하는 것이 바람직하다:
- 일반과세: 15.4%
- 세금우대종합저축: 9.5%
- 상호금융 저율과세: 1.4% (2025.12.31까지) → 5.9% (2026년) → 9.5% (2027년~)
- 비과세: 0%

## 2. 대출 이자 계산

### 공식: ✅ 정확

원리금균등(PMT 공식), 원금균등, 만기일시 3가지 상환방식 공식은 부동산계산기.com 및 한국주택금융공사, 교보생명, SC제일은행 계산기와 모두 일치한다.

### 수정 필요 사항

**체증식분할상환 추가 필요.** 부동산계산기.com은 4가지 상환방식(만기일시, 원금균등, 원리금균등, **체증식**)을 지원한다. 체증식은 보금자리론 등 일부 정책대출에서 사용되며, 초기 상환액이 적고 시간이 지날수록 증가하는 구조다. 주담대 종합 모듈을 만든다면 체증식도 포함하는 것이 실무적으로 유용하다.

## 3. LTV 계산

### 공식 구조: ✅ 기본 프레임 정확

### 수정 필요 사항 (3가지)

**(1) 소액임차보증금 최우선변제금액 차감 누락**

부동산계산기.com은 대출가능액 산출 시 다음 공식을 명시한다:

> 대출가능액 = 주택시세 × LTV% − 기존대출잔액 − 임차보증금 − **소액임차보증금 최우선변제금액**

소액보증금은 `(전체방수 − 1) × 소액보증금`으로 계산하며, 아파트는 방수 무관하게 1개로 적용 가능하다. MCI/MCG 보증부 대출은 소액보증금 차감 불요.

**(2) 기준일자 기능 필요**

부동산계산기.com은 "기준일자" 입력 기능을 제공하여, 6.27/9.7/10.15 대책 전후의 LTV를 비교할 수 있게 한다. 규정이 빈번히 변경되므로 기준일자별 분기 로직이 필요하다.

**(3) 현행 LTV 비율표 (2026년 4월 기준)**

| 구분 | 규제지역 (서울 전역 + 경기 12곳) | 비규제지역 |
|---|---|---|
| 무주택자 (일반) | **40%** | 70% |
| 생애최초 구매자 | **70%** (6.27 대책: 80%→70%) | 80% |
| 서민·실수요자 | **60%** (20%p 우대) | — |
| 처분조건부 1주택자 | 50% | 70% |
| 2주택 이상 | **0% (대출 불가)** | — |

**주택가격별 절대한도 (수도권·규제지역):**
- 15억원 이하: 최대 **6억원**
- 15억~25억원: 최대 **4억원**
- 25억원 초과: 최대 **2억원**
- 생활안정자금: 1주택자 최대 **1억원**, 2주택 이상 대출 불가

## 4. DTI 계산

### ✅ 공식·구조 모두 정확

부동산계산기.com도 동일한 DTI 공식을 사용한다:

> DTI = (주담대 연간 원리금상환액 + 기타대출 연간 이자상환액) / 연소득 × 100

**신DTI와의 구분:** 부동산계산기.com은 "신DTI"를 별도 제공한다. 신DTI는 기존 DTI와 달리 **기존 주담대의 원리금(원금+이자)까지** 포함하여 계산한다. 사용자 설계에서 "DTI"라 칭하는 것이 구DTI인지 신DTI인지 명확히 표기할 필요가 있다. 현재 사용자 설계의 입력항목(주담대 원리금 + 보유대출 연이자)을 보면 **신DTI에 해당**한다.

DSR 기준: 제1금융권 40%, 제2금융권 50%.

## 5. DSR / 스트레스 DSR — ❌ 가산금리 전면 수정 필요

### 기본 DSR 공식: ✅ 정확

> DSR = 모든 대출의 연간 원리금상환액 / 연소득 × 100

### 스트레스 DSR: ❌ 사용자 제시 수치가 현행과 불일치

**부동산계산기.com의 현행 스트레스 금리 (공시: 2025.12.31 기준, 현재 유효):**

| 지역 | 기본 스트레스 금리 | 비고 |
|---|---|---|
| **수도권·규제지역** 주담대 | **3.0%** | 10.15 대책 하한 상향 |
| **일반 (전국)** | **1.50%** | 3단계 기본 |
| **지방** 주담대 | **0.75%** | 26.6.30까지 유예 |

**대출유형별 적용비율 (3단계, 2025.7.1~):**

| 대출유형 | 적용비율 | 수도권·규제지역 실적용 | 지방 주담대 실적용 |
|---|---|---|---|
| **변동형** (금리변동주기 5년 미만) | **100%** | **3.0%p** | 0.75%p |
| **혼합형** (고정 후 변동) | **80%** | **2.4%p** | 0.60%p |
| **주기형** (5년+ 주기 고정) | **40%** | **1.2%p** | 0.30%p |
| **순수고정형** (만기 70%+ 고정) | **0%** | **0%** | 0% |

> **참고:** 2단계까지의 적용비율은 변동 100%, 혼합 60%, 주기 30%였으나, 3단계에서 혼합 80%, 주기 40%로 상향되었다.

### 사용자 제시 수치의 정오표

| 사용자 제시 | 정확한 현행 값 | 판정 |
|---|---|---|
| 변동형 +1.2%p | 수도권 **+3.0%p** / 지방 **+0.75%p** / 일반 **+1.5%p** | ❌ 2단계 수도권 값 |
| 혼합형 +0.45%p | 수도권 **+2.4%p** / 지방 **+0.60%p** / 일반 **+1.2%p** | ❌ 2단계 지방 값 |
| 주기형 +0.75%p | 수도권 **+1.2%p** / 지방 **+0.30%p** / 일반 **+0.6%p** | ❌ 지방 base rate와 혼동 |
| 미적용 | 순수고정형 0% → ✅ 이 옵션은 유지 | ✅ |

### 권고: 스트레스 DSR 입력 구조 재설계

사용자 설계의 "스트레스DSR(변동형/혼합형/주기형/미적용)" 4개 옵션은 유지하되, **지역 구분을 추가**해야 한다:

```
입력:
  - 지역: 수도권·규제지역 / 지방 / 기타
  - 금리유형: 변동형 / 혼합형 / 주기형 / 순수고정형
  
내부 계산:
  기본_스트레스금리 = 지역별 lookup (3.0% / 0.75% / 1.5%)
  적용비율 = 금리유형별 lookup (100% / 80% / 40% / 0%)
  실적용_가산금리 = 기본_스트레스금리 × 적용비율
  심사금리 = 실제금리 + 실적용_가산금리
```

스트레스 금리는 **반기별(6월/12월)** 금융위원회가 공시하므로, 동적 업데이트 구조가 필수적이다.

## 6. 중도상환수수료

### 공식: ⚠️ 단위 수정 필요

**사용자 설계:** `상환금액 × 수수료율 × (잔존기간/대출기간)` — "기간" 표기

**부동산계산기.com:** `중도상환원금 × 수수료율 × (잔존**일수** ÷ 대출기간)` — **일수(日數)** 기준 명시

KB국민은행도 동일하게 "잔존일수 = 대출원금 상환일에서 대출만기일까지 남은 일수"로 정의한다. **월 단위가 아닌 일 단위**로 계산해야 실무와 일치한다.

### 추가 반영 사항

- **2025.1.13 수수료율 대폭 인하:** KB 기준 부동산담보대출 1.2~1.4% → **0.58%**, 신용대출 0.6~0.7% → **0.02%**
- **면제기간:** 대출 실행일로부터 **3년** 경과 시 면제 (✅ 사용자 설계 정확)
- **연간 무수수료 상환 한도:** 매년 최초 대출금액의 **10%** 이내 상환 시 수수료 면제 (입주잔금대출은 30%)
- **고정금리 3년 이상 상품:** 별도 낮은 수수료율 적용 가능

## 7. 대출가능액 (LTV+DTI+DSR 종합)

### 부동산계산기.com과의 비교

사용자 설계의 "주담대 종합" 모듈은 `종합한도 = min(LTV, DTI, DSR 한도)`로 되어 있으며, 이 구조는 올바르다.

### 추가 반영 필요 사항

부동산계산기.com은 대출가능액 계산 시 다음을 추가로 고려한다:

1. **주택가격별 절대한도** (수도권·규제지역): 6억/4억/2억 cap
2. **소액임차보증금 최우선변제금액** 차감
3. **기준일자별 규정 분기** (6.27/9.7/10.15 대책)
4. **생활안정자금 한도** (1주택자 1억원, 2주택+ 불가)
5. **주택매매·임대사업자 LTV 0%** (수도권·규제지역)

따라서 종합한도 공식은:

> **종합한도 = min(LTV한도, DTI한도, DSR한도, 절대한도)**
